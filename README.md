# Sign Language Recognition – Transformer Model

Este repositorio contiene el sistema de reconocimiento de Lengua de Señas basado en **landmarks** y una arquitectura **Transformer**.

---

## Estructura relevante

```text
project/
│
├── transformer/
│   ├── data_/
│   ├── evaluation/
│   │   ├── inference.py
│   │   └── README.md
│   └── model/
│
└── outputs/
```

---

## ¿Cómo ejecutar inferencia?

La inferencia **se ejecuta desde la raíz del proyecto**.

Debes dirigirte al módulo:

```text
transformer/evaluation
```

Dentro de esa carpeta encontrarás:

* `inference.py`
* Un `README.md` con las instrucciones detalladas de ejecución
